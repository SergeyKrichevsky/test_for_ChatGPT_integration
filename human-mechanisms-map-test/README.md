# Human Mechanisms Map — Test Project

Тестовая папка для проекта **«Карта механизмов человека»**.

Цель этой тестовой зоны — проверить рабочий формат хранения сущностей, карточек и будущих связей в GitHub без риска для основной базы знаний.

## Текущая идея структуры

```text
human-mechanisms-map-test/
├── README.md
└── entities/
    └── energy-balance.md
```

## Принцип

- одна сущность = один Markdown-файл;
- YAML-блок в начале файла хранит структурные свойства карточки;
- основной текст файла хранит человеческое описание;
- категории и связи пока считаются рабочими гипотезами и могут меняться.

## Статус

Тестовый прототип. Не является финальной структурой проекта.
